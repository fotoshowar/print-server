# FotoShow Print Server - Linux/Pi Version

## Done ✅
- CUPS integration for printing
- Automatic installer script
- Systemd services (print + SSH tunnel)
- Auto-detect printer orientation
- Contact sheet/index printing (configurable)
- Preview modal before print
- Share via QR code
- Organized by date
- Thumbnails for fast loading

## Future Features
- Drag selection for photos
- Limit selection for index sheets
- Print queue management
- Photo editing (crop, filters)
- Multi-printer support
